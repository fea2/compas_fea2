********************************************************************************
Backends
********************************************************************************

COMPAS FEA2 supports multiple analysis backends through a plugin
architecture. Backends implement solver-specific functionality for input
file generation, analysis execution, and result extraction. Supported solvers
include Abaqus, ANSYS, SOFiSTiK, and OpenSEES.

For more information on configuring and developing backends, see
:doc:`/development/index`.
